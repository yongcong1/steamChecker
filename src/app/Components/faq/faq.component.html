Frequently asked questions
1. Why can't I see my stats?
A: Make sure your steam profile is public and game details is also public in Steam Settings
2. Why are some games not showing in game stats?
A: Game stats will only show games that has current player stats, in other words steam has no player statistics for that particular game.
3. Where are all these stats coming from?
A: All stats are retrieved using steam API with the exception of Player statistics which is retrieved using steamKit2 .NET library.
